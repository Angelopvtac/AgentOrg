# Changelog

## 1.0.0 — 2026-02-22

Initial public release.

### Features
- Dual PDF engine: WeasyPrint (HTML/CSS) and Typst (native typesetting)
- 25+ components: callouts, cards, columns, worksheets, metrics, stat cards, badges, charts, and more
- YAML front matter with cover page generation
- 3 built-in themes: default, dark, minimal
- Batch conversion of markdown directories
- Component reference via `md2pdf list-components`
- Typst engine: section banners, table of contents, presentation-style 16:9 layout
